# Wallpapers

Lege hier das Helldivers-Wallpaper als `helldivers-wallpaper.jpg` ab.

Da Helldivers urheberrechtlich geschützt ist, kann das Bild nicht direkt im Repository enthalten sein.

## Empfohlene Quellen

- Offizielle Helldivers-Artworks von [store.steampowered.com](https://store.steampowered.com/app/553850/HELLDIVERS_2/)
- Wallpaper-Seiten wie [wallhaven.cc](https://wallhaven.cc) (nach "Helldivers" suchen)
- Eigene Screenshots aus dem Spiel

## Dateiname

Das Wallpaper muss exakt so heißen:

```
helldivers-wallpaper.jpg
```

Oder passe den Pfad in `configs/xfce/xfce4-desktop.xml` an.

## Fallback-Wallpaper

Falls kein Helldivers-Wallpaper vorhanden ist, wird die XFCE-Standardfarbe (`#0a0e1a` dunkelblau) genutzt.
